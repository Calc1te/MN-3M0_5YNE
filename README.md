B4-rt 3n-der，或者data bar，是一个基于 **Tauri v2** 架构构建的跨平台、透明无边框桌面应用程序。具体内容如下：

* 后端: Rust (Tauri v2, Axum, LanceDB)
* 前端: React / Vite (Tailwind CSS)
* 协议层: JSON-RPC 2.0 / MCP (Model Context Protocol)

她会拿你不需要（或者仍然需要）的文件混在一起调酒，然后适当的挖苦你。

当前有以下功能：
- 和你有一搭没一搭的聊天
- 从你选定的目录找文件调酒
- 记住你的长期喜好
- 在你很久不理她的时候会自己找点事做

之后可能有的功能：

- 放音乐
